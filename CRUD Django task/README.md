# Zuri-Blog
A mini blog

Supports the following features for users. 
1. Registration 
2. Login 
3. Reset Password 
4. Create Post
5. Delete Post
6. Edit post
7. Comment on a post

Technologies used :-

Frontend: HTML5, CSS (Bootstrap4 + Custom CSS)
Backend: Django
Deployment: Heroku
